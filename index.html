Thrift使用初体验

  • 简单介绍
    • Apache Thrift是一个facebook建立的RPC框架，现在是一个Apache的顶级项目。Thrift允许通过一个跨语言的定义文件的方式定义数据类型和服务接口，这个文件作为RPC客户端和服务器通信的标准
  • 安装 thrift
    • 下载: wget http://apache.mirror.iweb.ca/thrift/0.9.2/thrift-0.9.2.tar.gz
    • 解压: tar -xvzf thrift-0.9.0.tar.gz
    • 配置: cd thrift-0.9.2 然后 sudo ./configure
    • 可能遇到的报错:configure: error: Bison version 2.5 or higher must be installed on the system!
    • 如何解决:brew install bison, brew link bison –force
    • 编译: sudo make
    • 安装: sudo make install
    • 验证是否安装成功: thrift –help

  • 使用Thrift

    • 编写thrift文件，并命名为hello.thrift，详细语法可参考官网，如何定义了一个名为Hello的thrift服务，服务中包含一个方法 helloString，方法有返回值类型，方法名，参数列表

      • 编译thrift文件，使用安装好的thrift编译器编译user.thrift生成python，java，c++代码
      • thrift –gen java hello.thrift
      • thrift –gen py hello.thrift
      • 生成结果: gen-java，gen-py目录，gen-java目录下有Hello.java文件

      • python使用上述编译好的python代码还需要添加python所需要的第三方依赖，使用pip istall thrift；同样的,java使用上述编译好的java代码需要添加maven依赖

  • 编写python服务端

    • 编写python服务端,需要将上面生成的python代码拷贝到你的项目里面，或者通过包引用的方式将目录添加进来，再引入到项目里面，java也是一样。将生成的python代码通过引用的方式引入到python环境。只需要在生成的gen-py目录下编写一个setup.py文件即可。然后执行 python setup.py install 命令。文件内容如下，使用import hello 验证是否安装成功

  • 删除
    • dd #删除一行

    • :%s/^(.*)n1/1$/ #删除重复行
    • :%s/^\s*//g# #删除行首多个空格
    • :%s/\s+$//g #删除行尾多个空格
    • dw #删除一个单词
  • 移动
    • gg #移动到第一行
    • G #移动到最后一行
    • 0 #移动到最首
    • $ #移动到行尾
    • w #移动到下一个单词的起始位置

MAC下三步搭建自己的hexo博客

Git安装

  • brew install git
  • 注册github，并成功使用，后续配置文件中要使用这个git仓库，所有发布的博客是托管在git服务器上的

Node.js安装

  • brew install node
  • 使用以下命令验证是否安装成功
    node -v;
    npm -v

Hexo安装和发布博客

  • npm install hexo -g #-g表示全局安装, npm默认为当前项目安装
  • hexo init xxx #新建博客目录，并且会在该目录下自动生成 _config.yml, - package.json文件和scaffolds, source, themes子目录
    • _config.yml为全局配置文件非常重要，配置博客的github地址，博客签名等信息
    • themes 为博客主题目录
    • source为本地存放博客的文件夹
    • package.json 应用程序数据，指明hexo的版本等信息
    • 其他目录和文件没研究过，但最好不要动
  • hexo g #根据当前目录下文件生成静态网页
  • npm install #初始化依赖
  • hexo s #启动服务器，可在本地http://0.0.0.0:4000/看到博客框架-

  • hexo n ‘我的第一篇文章’, 创建成功会有如下提示：INFO Created: ~/hexo/source/_posts/我的第一篇文章.md，博客内容的书写使用markdown语法，博客内容如下:

  • 找到对应下目录的文章并编辑，编辑好文章内容后依次执行如下命令就可以在 http://yourname.github.io/看到博客主页了

    • hexo g #生成
    • hexo s #本地预览
    • hexo d #发布到github
