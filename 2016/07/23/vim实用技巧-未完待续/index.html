vim实用技巧-未完待续

  • vim
  • 删除
    • dd #删除一行
    • x #删除一个字符
    • dG #删除当前行到最后一行
    • :g/^$/d #删除文件中所有空行
    • :%s/^(.*)n1/1$/ #删除重复行
    • :%s/^\s*//g# #删除行首多个空格
    • :%s/\s+$//g #删除行尾多个空格
    • dw #删除一个单词
  • 移动
    • gg #移动到第一行
    • G #移动到最后一行
    • 0 #移动到最首
    • $ #移动到行尾
    • w #移动到下一个单词的起始位置
    • ^ #移动到当前行的非空行行首
    • nG #移动到第n行
    • h #左移
    • j #下移
    • k #上移
    • l #右移
    • ctrl + f #向下滚动一屏
    • ctrl + b #向上滚动一屏
  • 显示
    • :set nu #显示行号
    • :set nonu #取消显示行号
  • 撤消
    • u #撤消上个命令的修改
  • 插入
    • i #进入编辑模式
    • o #在当前行的下一行进行插入
    • yy + p #复制当前行并粘贴到下一行
